Development of a Student Depression Prediction Model Based on Machine Learning with AlgorithmPerformance Evaluationv

Deskripsi Proyek

Artikel ini menyajikan upaya pengembangan sebuah model berbasis machine learning yang ditujukan khusus untuk memprediksi tingkat depresi di kalangan mahasiswa. Dalam penelitian ini, para peneliti memanfaatkan kumpulan data yang cukup besar, yakni 2.028 respons mahasiswa, yang memuat informasi akademik, demografis, dan hasil skor PHQ-9 (alat ukur depresi standar). Setelah melalui serangkaian tahapan pemodelan‚Äîmulai dari penyeimbangan data hingga penentuan fitur‚Äîenam algoritma berbeda kemudian dievaluasi. Secara mengejutkan, algoritma Support Vector Machine (SVM) menunjukkan kinerja paling superior. Akurasi model SVM ini melonjak drastis, dari awalnya 58,8% menjadi sekitar 99,5% setelah dilakukan penyesuaian parameter secara mendalam. Hasil ini menegaskan bahwa model prediksi ini sangat berpotensi menjadi kerangka kerja proaktif yang dapat diintegrasikan oleh pihak universitas untuk mendeteksi dan melakukan intervensi dini terhadap isu kesehatan mental mahasiswa, di mana faktor seperti gangguan tidur dan energi rendah teridentifikasi sebagai prediktor utama..
