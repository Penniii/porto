UI/UX Mobile App Design

Project Overview

This project is a mobile food delivery application designed to provide a seamless and visually engaging ordering experience. The main goal was to create an intuitive interface that allows users to easily browse categories, explore popular dishes, customize their orders, and complete purchases efficiently.

The design focuses on:
1. Clear visual hierarchy
2. Bold color identity to stimulate appetite
3. User-friendly navigation
4. Simplified ordering flow

The product detail page includes customization options such as spice level, toppings, ratings, and quantity selection to enhance user control and interaction. This project demonstrates my ability to combine visual aesthetics with functional usability principles in mobile interface design.
